Please following steps to see TRV report ( which has Excel Interactivity) from home.

1. Login to ENRON network using dialup or DSL or Cable Modem.
2. Go to http://remoteservices.enron.com
3. Select General Purpose cluster 2 - Notes R5 ( This is start terminal server on your home pc)
4. Login again to ENRON network using your Windows 2000 login and password. ( this might take few mints)
5. Go to Start->Program->IE to start IE browser
6. Go to http://trv.corp.enron.com
7. Reenter your windows Login.

Also you can do this testing at your work pc just follow above steps.
